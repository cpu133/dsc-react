Material-UI (http://www.material-ui.com/) frontend ONLY clone of DSC Alarm server project, https://github.com/denvera/dsc-node

### Installation
 - `git clone https://github.com/sj-louw/dsc-react`
 - `cd dsc-react`
 - `yarn install`
 - `yarn start`

http://localhost:3000
